# 如何查找你的 GitHub 用户名

## 方法 1: 登录 GitHub 查看

1. **登录 GitHub**：访问 https://github.com
2. **查看用户名**：
   - 登录后，点击右上角的**头像**
   - 或者查看浏览器地址栏
   - 或者访问你的个人资料页面，URL 格式为：`https://github.com/你的用户名`
   - 用户名显示在个人资料页面的顶部

## 方法 2: 通过邮箱查找

1. 访问：https://github.com/settings/emails
2. 可以看到与你的账号关联的邮箱和用户名

## 方法 3: 查看个人资料 URL

1. 登录 GitHub
2. 点击右上角头像 → 选择 "Your profile"（你的个人资料）
3. 查看浏览器地址栏，格式如：`https://github.com/你的用户名`
4. URL 中的 `你的用户名` 就是你的 GitHub 用户名

## 方法 4: 查看仓库创建页面

1. 访问：https://github.com/new
2. 创建仓库时，在 "Repository name" 上方会显示：`Owner: 你的用户名`

## 重要提示

⚠️ **GitHub 用户名** 和 **邮箱地址** 是不同的：
- **邮箱**：`dengxn@gmail.com` 
- **用户名**：可能是 `dengxn`，但也可能是其他名称（比如 `dengxn123`、`xdeng` 等）
